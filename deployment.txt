========================================
LOCAL SETUP INSTRUCTIONS
========================================

PREREQUISITES:
- Node.js (v14 or higher)
- MongoDB Atlas account
- Gmail account with App Password for email notifications

STEP 1: INSTALL DEPENDENCIES
-----------------------------
Backend:
  cd backend
  npm install

Frontend:
  cd frontend
  npm install

STEP 2: CONFIGURE ENVIRONMENT VARIABLES
----------------------------------------
Backend (.env in backend/):

MONGO_URI=mongodb+srv://username:password@cluster.mongodb.net/eventmanagement
JWT_SECRET=your_random_secret_key_minimum_32_characters
EMAIL_USER=your-gmail@gmail.com
EMAIL_PASS=your-16-character-app-password
PORT=5000

Frontend (.env in frontend/):

REACT_APP_API_URL=http://localhost:5000/api

STEP 3: RUN LOCALLY
-------------------
Start backend:
  cd backend
  npm start
  (Runs on http://localhost:5000)

Start frontend:
  cd frontend
  npm start
  (Opens at http://localhost:3000)

MONGODB ATLAS SETUP (if needed)
========================================
1. Create free account at https://www.mongodb.com/cloud/atlas
2. Create cluster and database user
3. Whitelist IP: 0.0.0.0/0
4. Get connection string and add to backend/.env as MONGO_URI

GMAIL APP PASSWORD SETUP (if needed):
1. Enable 2-Step Verification on Gmail
2. Generate App Password: Google Account > Security > App passwords
3. Add to backend/.env as EMAIL_PASS
1. Push code to GitHub repository
2. Connect GitHub repo to Render/Railway
3. Set build command: npm install
4. Set start command: node server.js
5. Set environment variables:
   - MONGO_URI=<your-mongodb-atlas-connection-string>
   - JWT_SECRET=<your-jwt-secret>
   - EMAIL_USER=<your-email>
   - EMAIL_PASS=<your-app-password>
6. Deploy

MONGODB ATLAS SETUP:
1. Create account at https://www.mongodb.com/cloud/atlas
2. Create new cluster (free tier available)
3. Add database user with password
4. Whitelist IP (0.0.0.0/0 for all IPs in production)
5. Get connection string from "Connect" button
6. Replace password in connection string
7. Use this as MONGO_URI environment variable
